Partion & Setting

Partioning

RedHat 권고 파티션 설정을 보고 해도 상관 없음.

  1. /boot

    500M(장치유형 : 표준 파티션, 파일시스템 : ext4)

  2. swap

    16G(해당 PC는 16G Ram 장착, 장치유형 : 표준 파티션, 파일시스템 : swap)

  3. /home

    100G(라이브리러등 이것저것 설치하다 보면 귀찮아서 걍 100G, 장치유형 : LVM, 파일시스템 : ext4)

  4. /

    나머지 용량 전부 때려 박는다(장치유형 : LVM, 파일시스템 : ext4)
